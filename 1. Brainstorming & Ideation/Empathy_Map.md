# Empathy Map — EduGenie AI Assistant

**Persona:** "Ananya" — University Student / Self-Learner (Role / User Type)

| SAYS | THINKS |
|---|---|
| "I don't have time to reread this whole chapter." | "There has to be a faster way to actually understand this." |
| "Can someone just quiz me on this right now?" | "I'm not sure I actually learned anything if no one tests me." |
| "My internet keeps cutting out during exam week." | "What if the tool stops working right when I need it most?" |
| "I don't want to pay for five different AI apps." | "One tool that does everything would save me money and time." |

| DOES | FEELS |
|---|---|
| Switches between 3-4 different apps/tabs to explain, summarize, and quiz herself | Overwhelmed by fragmented tools and repeated context-switching |
| Pastes long paragraphs into chatbots and waits for slow cloud responses | Anxious about running out of free API credits mid-session |
| Manually writes practice questions from memory before a test | Under-confident about whether she covered the right concepts |
| Studies late at night on patchy hostel Wi-Fi | Frustrated when a tool fails without an offline fallback |

**Key Insight:** Ananya doesn't need more AI tools — she needs *one* consistent, always-available assistant that compresses her entire study loop (read → understand → test → plan) into a single low-friction workspace. This directly motivated EduGenie's hybrid local + cloud design.
